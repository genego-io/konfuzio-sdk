Konfuzio SDK¶

The Konfuzio Software Development Kit (SDK) provides the tools to work with the data layer used by Konfuzio software. Using the SDK you can communicate with the Konfuzio App and use the data structure in your projects.

Quickstart¶

  1. pip install konfuzio-sdk

  2. Register online for free

  3. cd <your_working_directory>

  4. konfuzio_sdk init
