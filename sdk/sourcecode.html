konfuzio_sdk.api.post_document_annotation(document_id: int, start_offset: int, end_offset: int, label_id: int, template_id: int, accuracy: float, revised: bool = False, is_correct: bool = False, section=None, define_section=True, session=<requests.sessions.Session object>)¶

Add an annotation to an existing document.

For the section definition, we can: - define the section id where the annotation should belong (section=x (int), define_section=True) - pass it as None and a new section will be created (section=None, define_section=True) - do not pass the section field and a new section will be created if does not exist any or the annotation will be added to the previous section created (define_section=False)

Parameters

  • document_id – ID of the file

  • start_offset – Start offset of the annotation

  • end_offset – End offset of the annotation

  • label_id – ID of the label.

  • accuracy – Accuracy of the annotation

  • revised – If the annotations are revised or not (bool)

  • session – Session to connect to the server

Returns

Response status.
